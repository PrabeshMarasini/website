IC Firewall

IC Firewall Github Repository


This project aims to develop a Firewall that can connect to multiple devices, including servers, to monitor and manage network traffic.

Stage 1: Packet Tracing and Filtering

    In the initial stage, the focus will be on creating a packet tracing system. This will involve:

  • Tracking incoming and outgoing packets on the device in real time.
  • Logging source and destination ports associated with each packet, allowing for port-specific monitoring.
  • Extracting IP addresses from packet headers to identify where packets are coming from and where they are going.
  • Blocking packets from suspicious IP addresses that may pose security threats.

Root access will be required to enable low-level network monitoring and packet filtering functionalities. Development will initially target Linux platforms.
